# 使用说明

## 如何阅读新闻

1. 在左侧目录栏中选择年份
2. 展开对应的月份
3. 点击日期查看当天的新闻汇总

## 搜索新闻

使用页面顶部的搜索框（🔍 图标）可以搜索新闻内容。

## 目录结构说明

新闻按以下层级组织：

```
src/
├── SUMMARY.md           # 总索引
├── README.md            # 首页
├── ABOUT.md             # 本页面
└── 2026/                # 按年份分类
    ├── SUMMARY.md
    └── 01/              # 按月份分类
        ├── SUMMARY.md
        └── 31.md        # 具体日期的新闻
```

## 新闻分类

每篇新闻汇总包含以下分类：

- **时政新闻** - 国内政治、政策等
- **国际新闻** - 国际重大事件
- **经济新闻** - 经济、金融、市场等
- **社会新闻** - 社会热点、民生等
- **科技新闻** - 科技、互联网等
- **体育娱乐** - 体育、娱乐新闻等

## 常见问题

**Q: 新闻多久更新一次？**
A: 每天上午 9:00（北京时间）自动更新。

**Q: 新闻数据来源是什么？**
A: 来源于中国大陆各大主流新闻网站。

**Q: 如何分享单篇新闻？**
A: 点击新闻标题，复制浏览器地址栏的 URL 即可分享。

**Q: 可以下载新闻吗？**
A: 可以，点击右侧的 ⬇️ 图标下载 PDF 格式。
