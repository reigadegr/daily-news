# 每日新闻汇总

欢迎来到每日新闻汇总网站！

这里汇集了从中国大陆各大新闻网站获取的每日新闻，涵盖时政、经济、国际、社会、科技等多个领域。

## 目录结构

新闻按 **年份 → 月份 → 日期** 的层级结构组织：

```
2026年/
├── 1月/
│   ├── 31日
│   ├── 30日
│   └── ...
├── 2月/
│   └── ...
└── ...
```

## 最新新闻

最新新闻请查看：[2026年1月31日](./2026/01/31.md)

## 数据来源

新闻来源于以下中国大陆主流新闻网站：

- 澎湃新闻
- 网易新闻
- 央视新闻
- 新华网
- 人民网
- 环球网
- 等等

## 更新时间

本网站每日自动更新，新闻采集时间约为每天上午 9:00（北京时间）。

## 关于本项目

本项目使用 OpenClaw 自动化工具每天采集新闻，并推送到 GitHub 仓库。

- GitHub 仓库：[https://github.com/reigadegr/daily-news](https://github.com/reigadegr/daily-news)
- 本网站：[https://reigadegr.github.io/daily-news](https://reigadegr.github.io/daily-news)
