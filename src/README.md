## 平台介绍

### GitHub
全球最大的开源代码托管平台，拥有最活跃的开源社区。我们收集 GitHub 每日/每周/每月的 Trending 项目。

### Gitee
中国最大的代码托管平台，拥有大量优质的中文开源项目。我们收集 Gitee 的 GVP (Gitee Most Valuable Project) 和热门项目。

### GitCode
CSDN 旗下的代码托管平台，专注于中文开发者社区。我们收集 GitCode 的热门推荐项目。

### AtomicGit
新兴的代码托管平台，专注于创新和前沿技术。我们收集 AtomicGit 上的特色项目。
